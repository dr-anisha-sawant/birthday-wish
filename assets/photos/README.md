# Photos

Drop your photos here, named `photo-01.jpg`, `photo-02.jpg`, etc.

The gallery reads them via the `photos` array in [`js/gallery.js`](../../js/gallery.js) — edit that array to add captions, dates, or change filenames.

Recommended:
- 1080–1600px wide, JPG or WebP, compressed (under ~250 KB each)
- Aspect ratio doesn't matter — gallery uses `object-fit: cover`
